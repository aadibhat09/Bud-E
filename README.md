## Setup

### Getting a Gemini API Key
1. Visit [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Sign in with your Google account
3. Create a new API key
4. Copy the key and paste it into the extension's settings

### Installing the Extension
1. Open Chrome/Edge and navigate to `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the Bud-E folder

## Usage
1. Click on the Bud-E extension icon in your browser
2. Add productive websites to your whitelist (e.g., github.com, docs.python.org)
3. Save your Gemini API key in the AI Productivity Coach section
4. Click "Get Productivity Suggestions" to receive personalized tips
5. Watch your Bud-E grow as you stay productive!
